  RAMDISK.SYS (n[E][F] [file])|OFF

This resident command sets up an
emulated disk drive using extended RAM
(up to 1MB).

The "n" parameter is the number of the
Ramdisk drive.
The "F" option will cause the RamDisk
to be formatted.
The "E" option will cause that it will
not use 130XE compatible banks.
OFF removes the driver, if it was the
last driver loaded.

The configuration file can be used to
specify values for PORTB to be used for
the Ramdisk.

Command requires 48k main memory.
