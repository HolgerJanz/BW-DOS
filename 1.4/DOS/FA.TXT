Fast Assembler 1.8 23-11-2023.
Assembler for SpartaDOS X and BW-DOS.
 src - ATASCII source file
 obj - target binary object file
The default extensions (ext) are .ASM
and .OBJ. The default obj name is the
src name.
 /L list combined source
 /S print summary, e.g. block and
    label count, memory (hexadecimals)
    code-stat_data-last dyn_data-last
 /E sets compile errors to system
    errors starting from 224($E0)
 /Y writes symbol table to file
 /Q quite, no line progress output
